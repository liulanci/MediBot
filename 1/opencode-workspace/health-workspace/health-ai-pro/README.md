# Health AI Pro - 全科医疗健康评估系统

## 项目概述

这是一个综合性的医疗健康AI助手，集成了多种健康评估量表、AI问答系统、健康数据分析等功能。

## 功能模块

### 1. AI 问答系统
- 智能医疗问答
- 可视化/音频通知提醒
- 基于证据的回答

### 2. 健康评估量表
- **精神科**: HAMD, HAMA, BPRS, EPQ, BRMS, SCL-90, MMPI, EPDS, RCADS, MMSE
- **心内科**: HYHA, SAQ
- **疼痛科**: VAS, ODI
- **骨科**: KOOS, GCS, MRS
- **妇产科**: CAT, PVC, PFDI-20
- **儿科**: DDST
- **肿瘤科**: EORTC QLQ-C30
- **综合**: ADL, MBTI

### 3. 健康数据分析
- 血压监测
- 血糖监测
- BMI计算
- 生活习惯评估
- 医学报告解读
- 单位换算

### 4. 个性化设置
- LLM模型配置
- 主题切换
- 用户画像管理

## 技术栈

- 前端: HTML5, CSS3, JavaScript (ES6+)
- 后端: Node.js
- AI: 兼容多种LLM API
