## 🚀 部署步骤

### 1. 同步到 GitHub

#### 方法一：使用脚本（推荐）

1. 双击 `deploy-to-github.bat`
2. 等待完成
3. 完成！

#### 方法二：手动同步

1. 复制所有文件到 GitHub 仓库
2. 提交：`git add . && git commit -m "🎉 医智宝 v1.0.0"`
3. 推送：`git push -u origin main`

### 2. Vercel 部署

1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 导入 `MediBot` 仓库
4. 点击 "Deploy"

### 3. 完成！

- 🌐 网站上线
- 📱 可在所有设备访问
- 🤖 AI 按钮可拖动
